# Unified Catalog

[What is the Microsoft Purview Unified Catalogue? Get control of your data, Microsoft Mechanics](https://www.youtube.com/watch?v=eGo9mJldifo) 

The Microsoft Purview Unified Catalogue is a centralised, searchable repository that acts as the foundation for data governance, helping organisations discover, understand, and manage their data across hybrid and multi-cloud environments. \\

**It allows users to register, classify, annotate, and govern data** by providing a single, integrated platform for data discovery and access control. 

The Unified Catalog connects to various data sources to provide a comprehensive, bird's-eye view of an organization's entire data estate, including: 

- Microsoft services like Azure and Fabric,   
- as well as non-Microsoft platforms like Databricks and Snowflake

# Compliance Manager

Microsoft Purview Compliance Manager is a cloud-based tool within the Microsoft Purview console that helps organisations manage their compliance posture by **assessing risk, providing actionable guidance, and tracking progress against regulations and industry standards like GDPR, ISO, and NIST**. 

It offers a centralised platform to **implement control actions, assign responsibilities, and generate reports for auditors,** streamlining the process of achieving and **maintaining regulatory compliance across Microsoft 365** and other cloud services.
