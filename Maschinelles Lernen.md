#### den Spieß umkehren or umdrehen

**Die Grundidee mit** maschinelles Lernen ist, dass **die Vorgehensweise,** bei der [ein Programm](https://www.collinsdictionary.com/dictionary/german-english/programm) (n) üblicherweise funktioniert, **[komplett](https://www.collinsdictionary.com/dictionary/german-english/komplett) [umgedreht](https://www.collinsdictionary.com/dictionary/german-english/umdrehen)** wird. [**Stattdessen**](https://www.collinsdictionary.com/dictionary/german-english/stattdessen) **festgelegte Regeln auf** Daten anwenden, **wird den [Spieß](https://www.collinsdictionary.com/dictionary/german-english/spiess) umdreht** und de**m** Computer die Daten und das Resultat geben werden, um den Computer **die Regeln** selbst **herausfinden zu** lassen.

Das ist **im Grund** maschinelles Lernen\!

[03.Einfuehrung in Konvolutionale Neuronale Netzwerke (ML Zero to Hero, Teil 3\)](https://www.youtube.com/watch?v=ZdTang-alP4) 

Es gab hier für ein [**Beschränkung**](https://www.collinsdictionary.com/dictionary/german-english/beschrankung) (n)   
die Anzahl von etwas / die Ausgaben / die Bedeutung eines Wortes [**beschränken**](https://www.collinsdictionary.com/dictionary/german-english/beschranken)**.**  
**die Rede / die Freiheit / ( den [Aufsatz](https://www.collinsdictionary.com/dictionary/german-english/aufsatz) (m) \= die Themen ) / die Diskussion [beschränken](https://www.collinsdictionary.com/dictionary/german-english/beschranken).**  
[**die Verwendbarkeit**](https://www.collinsdictionary.com/dictionary/german-english/verwendbarkeit) **\= [die Nutzbarkeit](https://www.collinsdictionary.com/dictionary/german-english/nutzbarkeit) / [die Brauchbarkeit](https://www.collinsdictionary.com/dictionary/german-english/brauchbarkeit) / [die Anwendbarkeit](https://www.collinsdictionary.com/dictionary/german-english/anwendbarkeit) stark beschränken.**

Das Build [**zentrieren**](https://www.collinsdictionary.com/dictionary/german-english/zentrieren).  
Das Build de**s** [Schuh](https://www.collinsdictionary.com/dictionary/german-english/schuh)**s** (m) muss nicht nur [**zentriert**](https://www.collinsdictionary.com/dictionary/german-english/zentrieren), sonder muss es **auch das** einzige **Objekt** in das Build sein.

Das Programm würde nicht **in der Lage** sein, eine**n** Schuh in anderen Bilder zu [**identifizieren**](https://www.collinsdictionary.com/dictionary/german-english/identifizieren), weil sie sehr anders in der Datensatz **aussehen** als die, die für das Trainieren de**s** Programm**s** und die Schaffung de**s** Model**s  gerbraucht wurden.**

[**Die Identifizierung**](https://www.collinsdictionary.com/dictionary/german-english/identifizierung) **und [die Klassifizierung](https://www.collinsdictionary.com/dictionary/german-english/klassifizierung)** von **[Objekt](http://v)e** (n) die nicht direkt in der Datensatz [**auftreten**](https://www.collinsdictionary.com/dictionary/german-english/auftreten) oder die ncht ganz **zentriert** in das Bild sind kann viel schwieriger oder sogar auch unmöglich sein, wenn nur **einfache Trainingsmodelle** verwendet werden. **Dafür** müssen wir nutzen, was wir als **konvoluzionales neuronales Netzwerk nennen**.

Ein [**Filter**](https://www.collinsdictionary.com/dictionary/german-english/filter) **(n)  zu** eine**m** Bild **anwenden.**  
Ein [**Filter**](https://www.collinsdictionary.com/dictionary/german-english/filter) **i**st eigentlich nur **[eine Matrix](https://www.collinsdictionary.com/dictionary/german-english/matrix)** oder **[Multiplikator](https://www.collinsdictionary.com/dictionary/german-english/multiplikator) (m)**.  
Ein [**Filter**](https://www.collinsdictionary.com/dictionary/german-english/filter) **entfernt** von das Bild  üblicherweise allen **Besonderheiten** außer einer  und dass [**betont**](https://www.collinsdictionary.com/dictionary/german-english/betone) sie stark  in [**das Resultat**](https://www.collinsdictionary.com/dictionary/german-english/resultat).   
**Jedes Filter** **legt viel größer Gewicht auf** einer Besonderheit als die andere**n.**  
Filtern sind üblicherweise mit **das Pooling** **kombiniert**, damit werden **viele Buildelemente** des Bildes [**zusammengefasst**](https://www.collinsdictionary.com/dictionary/german-english/zusammenfassen)**.** Die **Unter[gruppe](https://www.collinsdictionary.com/dictionary/german-english/gruppe)n** werden dann durch das Filter, das [**verstärkt**](https://www.collinsdictionary.com/dictionary/german-english/verstarken) auf das Build **auswirkt**.  
Ein Polling von 2  mal 2  Pizel z.B. [**gruppiert**](https://www.collinsdictionary.com/dictionary/german-english/gruppieren)

**Die Idee hinter** das konvoluzionale neuronale Netzwerk ist **es**, dass man die Bilder **vor** de**m** Trainier de**s** neuronale**s** Netzwer**s** [**filtert**](https://www.collinsdictionary.com/dictionary/german-english/filtern). Das Ziel de**s** Filter**s** ist die [**Besonderheit**](https://www.collinsdictionary.com/dictionary/german-english/besonderheit)en des Bildes **im Vorschein zu bringen**, so dass das maschinelles Lernen sie nützen kann während des Lernens  und später um die Objekte richtig zu **identifizieren**.

Pooling wird **eingesetzt** um [**die Größe**](https://www.collinsdictionary.com/dictionary/german-english/grosse) eine**s** Bilde**s zu reduzieren** z.B. wenn ein 2 mal zwei Pooling **auf** eine**m** Build angewendet wird, wird nur de**n** Pixel ausgewält, **der** de**n** höchest Wert hält, aus jede**r** [**Gruppe**](https://www.collinsdictionary.com/dictionary/german-english/gruppe) von 2 mal 2 Pixeln. **Das führt es zu** eine**m** neue**n** Bild, dass ein viertel de**r Größe** de**s** originale**n** Bilds ist, [**trotzdem**](https://www.collinsdictionary.com/dictionary/german-english/trotzdem) [**bleiben**](https://www.collinsdictionary.com/dictionary/german-english/haften-bleiben) die **Eigenschaften** [**halten**](https://www.collinsdictionary.com/dictionary/german-english/haften-bleiben).
